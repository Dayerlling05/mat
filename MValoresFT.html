Funciones trigonométricas en ángulos notables

Explicación

Se traza el lado terminal OP, con OP=r, para cada uno de los ángulos 0°, 90° y 180° en el plano cartesiano, como sigue:

En todos los casos, los lados terminales quedan sobre los ejes de coordenadas, a estos ángulos se les llama cuadrantales. Se sustituyen los valores correspondientes para θ, x, y y ren senθ = y/r, cosθ = x/r y tanθ = y/x, se puede completar la tabla así:

θ 0° 90° 180°
sen θ sen0° = 0/r = 0 sen90° = r/r = 1 sen180° = 0/r = 0
cos θ cos0° = r/r = 1 cos90° = 0/r = 0 cos180° = -r/r = -1
tan θ tan0° = 0/r = 0 tan90° = r/0 IND tan180° = 0/-r = 0

Para encontrar los valores de las funciones trigonométricas para los ángulos especiales, se traza una circunferencia de radio r y el lado terminal del ángulo especial, se determinan las coordenadas del punto de intersección P y se sustituyen dichos valores en la definición de cada una de las funciones trigonométricas.

Ejercicios

Determine los valores senθ , cosθ y tanθ para los ángulos 270° y 360°. ¿Con qué valores coinciden?

  • b) θ = 360°
