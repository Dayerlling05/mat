Multiplicación de binomio por trinomio de forma horizontal

Explicacion

Efectúe el producto (x + 2)(x + y + 3) de forma horizontal.

Para efectuar el producto (x + 2)(x + y + 3) se realiza lo siguiente:

1. Se multiplica cada término del binomio x+2 por el trinomio x + y + 3:

2. Se aplica la propiedad distributiva en los productos indicados que se obtuvieron en el paso anterior,

Para calcular el producto de un binomio por un trinomio, se multiplica cada término del binomio por cada término del trinomio.
